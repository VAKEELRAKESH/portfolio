### **Repository Setup Steps**

**Step 1: Create Repository**
- Go to GitHub → New Repository
- Name: `portfolio` (or `vakeelrakesh.github.io` for custom domain)
- Description: "Professional portfolio showcasing workflow automation and mechanical engineering expertise"
- Public repository
- Initialize with README

**Step 2: Upload Files**
- Upload `index.html`, `style.css`, `profile-photo.jpg`, and `README.md`
- Commit message: "Initial portfolio upload"

**Step 3: Enable GitHub Pages**
- Go to repository Settings → Pages
- Source: Deploy from a branch
- Branch: `main` → `/root` → Save
- Wait 2-3 minutes for deployment

**Step 4: Verify**
- Your portfolio will be live at: `https://[your-username].github.io/portfolio`
- Test all links and responsiveness
